# rockies-projects
Analysis of 2025 Rockies vs rest of MLB in a few metrics, and Batting Lineup Optimizer with 2026 Projections

Instructions to Run Code Files

Before running the files, you can clone the repo and install all of the needed the libraries for both projects in the terminal directly with this command:

pip install pybaseball pandas numpy tqdm matplotlib seaborn scipy

Then, in order to run the COL vs MLB analysis, you just run this command in the terminal from the rockies-projects folder: 
python rockies_2025_analysis.py

If you want to run the batting lineup optimizer, you navigate to the rockies-projects/rockies_lineup_optimizer folder and run: 
python main.py
